# Networking Basics Using C

This is a beginner-level menu driven C program created to understand
basic networking concepts.

## Topics Covered
- What is Networking
- Types of Networks
- TCP/IP
- Types of Communication
- Network Topology
- Transmission Modes
- Network Connectors (RJ45, SC, LC, ST)

## Language Used
- C

## Level
Beginner (First Semester Project)
